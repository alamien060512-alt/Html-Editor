# HTML Editor

HTML Editor is an Android application designed for creating and editing HTML files directly on your device.

## Features

- Create new HTML files
- Edit existing HTML documents
- Syntax-friendly code editing
- Save and manage projects locally
- Mobile-friendly workflow

## Installation

1. Download the APK file.
2. Enable installation from unknown sources if required.
3. Install the APK on your Android device.
4. Launch the app.

## Usage

1. Open the app.
2. Create a new HTML file or open an existing one.
3. Edit your code.
4. Save your changes.
5. Preview or export your work if supported by the app version.

## Requirements

- Android device
- Sufficient storage space for projects
